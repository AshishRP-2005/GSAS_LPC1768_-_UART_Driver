## Memory Mapped Input Output - MIMO
- This basically means that all the hardware peripheral registers are connected to a memory address
- By being mapped to an address it allows for the hardware to be accessed easily by the software
- Registers are small storage units that basically store bits and use bit manipulation to turn ON or OFF any hardware
